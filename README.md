# A RESTful API built using ASP.NET Core Web API
Built following the onion architecture and the CQS (command-query separation) principle.

# Details
- SQL Server for Database
- Entity Framework Core for DB access
- Identity for User handling and Role-based access control (RBAC)
- Bogus for generating fake data
- JSON Web Tokens (JWT) for authorization
